Enrutamiento de Angular

El presente proyecto consiste en demostrar de una forma basica el funcionamiento SPA de Angular. Por medio del sistema de enrutamiento,la creacion y reutilizacion de componentes se le permite al usuario desarrollar una aplicación de una sola página con múltiples vistas facilitando la navegación entre ellas. El usuario puede cambiar entre estas vistas sin perder el estado y las propiedades de la aplicación.
